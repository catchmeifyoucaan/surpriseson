# `surprisebot tui`

Open the terminal UI connected to the Gateway.

Related:
- TUI guide: [TUI](/tui)

## Examples

```bash
surprisebot tui
surprisebot tui --url ws://127.0.0.1:18789 --token <token>
surprisebot tui --session main --deliver
```
