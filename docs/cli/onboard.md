# `surprisebot onboard`

Interactive onboarding wizard (local or remote Gateway setup).

Related:
- Wizard guide: [Onboarding](/start/onboarding)

## Examples

```bash
surprisebot onboard
surprisebot onboard --flow quickstart
surprisebot onboard --mode remote --remote-url ws://gateway-host:18789
```
