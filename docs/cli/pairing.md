# `surprisebot pairing`

Approve or inspect DM pairing requests (for channels that support pairing).

Related:
- Pairing flow: [Pairing](/start/pairing)

## Commands

```bash
surprisebot pairing list whatsapp
surprisebot pairing approve whatsapp <code> --notify
```
